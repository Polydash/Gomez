			COMPILATION SOURCES
			
Pour compiler les sources, il est nécessaire d'avoir les paquets suivants
installés : 

	SDL2
	SDL2-image
	SDL2-ttf
	
Ces paquets sont disponibles au format .deb, dans le répertoire "Packages".
Ceci fait, les commandes suivantes sont disponibles : 

	make 				(Compilation en mode DEBUG)
	make MODE=RELEASE	(Compilation en mode RELEASE)
	make documentation	(Générer la documentation)
	make clean			(Nettoyer le répertoire du projet)
